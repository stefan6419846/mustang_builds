## Mustang builds

This repository provides an automated way to build snapshot JAR files for [Mustang project](https://github.com/ZUGFeRD/mustangproject).
Mustang is a Java-based CLI tool to verify electronic invoices, mostly regarding the standards EN 16931, ZUGFeRD/Factur-X and XRechnung.

### Structure

`.github` contains the GitHub Actions workflow files for building the JAR files.
